Research Data and Code Availability

Article: PHLDA3 marks kinase-driver-defined thyroid carcinoma: an integrated multi-omics and single-cell analysis
Author: Xia Wu

All data analysed in this study are publicly available from TCGA/GDC, UCSC Xena, GEO, cBioPortal, the Human Protein Atlas, ENCORI/starBase, L1000CDS2, oncoPredict/GDSC2, and CZI CELLxGENE Census, as stated in the manuscript Data Availability Statement.

Analysis code, software versions, processed result tables, and figure-generating scripts are available at:
https://github.com/candicewu0515/PHLDA3-THCA

This README is supplied as a Research Data/Code availability note. Raw public datasets and figure image files are not bundled here.
